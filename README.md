Since I no longer use this code or have BotVerse running, so I thought why not give out the source code to the people that would like to make their own rp bot that everyone can use for free. This source code is just a template for people that would like to start off somewhere and go from there. 

I hope you all enjoy this source code and can learn alot from this and can create other RP bots where everyone can use it for free.
